# KapanLibur

iOS version of [kapanlibur.krisdev.my.id](https://kapanlibur.krisdev.my.id/) — Indonesian national holiday information.

## Features

- Today's holiday display
- Upcoming holidays with countdown timer
- Leave suggestions

## Tech Stack

- SwiftUI

## API

Fetches holiday data from the original KapanLibur API.

## Development

Requires Xcode and iOS 15+.
